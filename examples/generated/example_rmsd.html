Plotting RMSD driftÂ¶

In [1]: import mdtraj as md

Find two files that are distributed with MDTraj for testing purposes – we can us them to make our plot

In [2]: import mdtraj.testing

In [3]: crystal_fn = mdtraj.testing.get_fn('native.pdb')

In [4]: trajectory_fn = mdtraj.testing.get_fn('frame0.xtc')

Load up the trajectories from disk

In [5]: crystal = md.load(crystal_fn)

In [6]: trajectory = md.load(trajectory_fn, top=crystal)  # load the xtc. the crystal structure defines the topology

In [7]: print trajectory
<mdtraj.Trajectory with 501 frames, 22 atoms>

Let’s also try loading only the heavy atoms from disk, because calculating RMSD with exchangeable hydrogen atoms is generally not a good idea

In [8]: heavy_atoms = [atom.index for atom in crystal.topology.atoms if atom.element.symbol != 'H']

In [9]: heavy_crystal = md.load(crystal_fn, atom_indices=heavy_atoms)

In [10]: heavy_trajectory = md.load(trajectory_fn, top=crystal, atom_indices=heavy_atoms)

In [11]: print heavy_trajectory
<mdtraj.Trajectory with 501 frames, 10 atoms>

Format the data for the RMSD calculation by building an RMSDCache.

In [12]: crystal_cache = md.rmsd_cache(crystal)

In [13]: trajectory_cache = md.rmsd_cache(trajectory)

In [14]: crystal_heavy_cache = md.rmsd_cache(heavy_crystal)

In [15]: trajectory_heavy_cache = md.rmsd_cache(heavy_trajectory)

In [16]: rmsds_to_crystal = trajectory_cache.rmsds_to(crystal_cache, 0)

In [17]: heavy_rmds_to_crystal = trajectory_heavy_cache.rmsds_to(crystal_heavy_cache, 0)

Plot the results

In [18]: import matplotlib.pyplot as pp

In [19]: pp.figure();

In [20]: pp.plot(trajectory.time, rmsds_to_crystal, 'r', label='all atom');

In [21]: pp.plot(heavy_trajectory.time, heavy_rmds_to_crystal, 'b', label='heavy atom');

In [22]: pp.legend();

In [23]: pp.title('RMSDs to crystal');

In [24]: pp.xlabel('simulation time (ps)');

In [25]: pp.ylabel('RMSD (nm)');
